- [ ] ادخل بريد وكلمة مرور أي دور
- [ ] تحقق من الدخول الناجح
- [ ] تحقق من ظهور لوحة التحكم الصحيحة

### 2. اختبار لوحة تحكم المريض
- [ ] اعرض الصحة العامة
- [ ] احمِّل صورة طبية
- [ ] ارسل رسالة للطبيب

- [ ] اعرض قائمة المرضى
