                    ❓ TROUBLESHOOTING GUIDE
═══════════════════════════════════════════════════════════════════════════════

PROBLEM: Backend won't start (EADDRINUSE: port 5000 in use)
SOLUTION: 
    lsof -i :5000
    kill -9 <PID>
    npm start

PROBLEM: Frontend won't compile
SOLUTION:
    rm -rf node_modules package-lock.json
    npm install
    npm start

PROBLEM: Can't connect to backend from frontend
SOLUTION:
    1. Check backend is running: curl http://localhost:5000/api/health
    2. Check CORS in backend/server.js is configured
    3. Check firewall settings

PROBLEM: Database has no data
SOLUTION:
    cd backend
    npm run seed

PROBLEM: Tests failing with authentication errors
SOLUTION:
    Make sure you ran: cd backend && npm run seed
    Check database file exists: backend/noura.db
